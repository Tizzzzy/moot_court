================================================================================
KEY DESIGN DECISIONS
================================================================================

1. Evidence Summary Approach
   - Full files uploaded once (turn received) for cost efficiency
   - Text summary passed to all subsequent turns
   - Maintains context while reducing API costs

2. Defendant Evidence Confidence Levels
   - LLM instructed to vary confidence (certain/probable/uncertain)
   - Prevents all claims treated as equally strong
   - Realistic scenario modeling

3. Feedback Display Timing
   - Triggered after objection check, only if statement passes
   - Non-blocking - user can skip reading
   - Informational only, not in official record

4. Monitoring Severity Filtering
   - Only displayed if severity > 'none'
   - Avoids cluttering output for clean statements
   - Educational without being intrusive
